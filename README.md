## Amazon S3 Backup

Use Amazon S3 Backup for Frappe and ERPNext

#### License

MIT


#### How to Install


1) Install boto and filechunkio packages

    a) pip install boto
    b) pip install filechunkio

2) bench get-app s3_backup https://github.com/ccfiel/s3_backup.git
3) bench install-app s3_backup
